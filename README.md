# 🌐 SpringWeb

A Spring-based web application demonstrating RESTful API development with Spring MVC. This project implements a Double Controller pattern to showcase clean separation of concerns in a Spring Web architecture.

## 🚀 Features

- **Spring MVC Framework** - Built on top of the Spring Web framework for robust HTTP handling
- **Double Controller Implementation** - Demonstrates best practices for organizing controllers in a Spring application
- **RESTful API Design** - Follows REST principles for clean and scalable API endpoints
- **Maven Build System** - Standard Maven project structure with dependency management
- **JUnit Testing** - Test-driven development approach with unit test coverage

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| Java | Core language |
| Spring Web | Web framework |
| Spring MVC | MVC architecture |
| Maven | Build & dependency management |
| JUnit | Unit testing |

## 🏃 Getting Started

### Prerequisites

- JDK 11 or higher
- Maven 3.6+

### Installation

```bash
# Clone the repository
git clone https://github.com/MuskaanSultana/SpringWeb.git

# Navigate to project directory
cd SpringWeb

# Build the project
mvn clean install

# Run the application
mvn spring-boot:run
```

## 🧪 Running Tests

```bash
mvn test
```
